Eye states

Toggle by setting data-state on the SVG root. Default if unset: alert.

StateMeaningUse when…
alert (default)Attentive, eyes open, glancing up.Idle and ready for input.
waiting Eyes closed in a content upward arch.Peaceful idle. Long pauses inside a single user turn.
sleep Eyes closed in a downward crescent.Long inactivity / low-power state. Agent paused.
think Eyes squeezed shut, chevrons pointing inward.Active processing. Knox is working on a response.
surprise Wide hand-drawn eyes, small pupils.Error, unexpected event, attention required.

Logo do / don't

DoDon't
Pair --knox-fur and --knox-ink from the listed combos. Mix colors from outside the primary/secondary/tertiary palette.
Place Knox on a surface listed for the chosen combo. Place classic Knox on --p-900 (ink and surface match — features disappear).
Keep stroke widths as defined in the SVG. Add drop shadows, gradients, or extra colors.
Set data-state to communicate agent state. Animate or transition the fox itself — state changes only.
