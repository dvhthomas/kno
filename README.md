# Kno

Personal agent harness. Run agentic workflows from a web browser; depend on it daily.

> Pre-implementation. **Kno-Lite v1 scope** per [ADR-0018](docs/adr/0018-kno-lite-scope.md). Daily-driver personal AI; not a corporate product. v1 workflows: default chat with memory, Flow Coach (GitHub flow metrics), KB-QA (cited answers from your writing). Full picture: [`docs/spec.md`](docs/spec.md).
